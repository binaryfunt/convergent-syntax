# Convergent syntax theme for Atom

A cooler & slightly punchier colour palette, similar to the [Brackets Convergent theme](https://github.com/binaryfunt/convergent-brackets-theme) (but different in a few ways).

## Installation

Search for "convergent-syntax" in the `Install` section of the Settings view (make sure to click `Themes` instead of `Packages`)
